# DevFlow-AI

DevFlow AI is an AI-powered developer productivity platform designed to help students and developers streamline learning, project building, and collaboration.

## Audience

- Students
- Beginner developers
- Open-source contributors

## Stack

- React frontend
- Node.js backend
- AI-focused recommendation and workflow automation endpoints

## Run locally

```bash
npm install
npm test
npm run build
npm start
```

Open `http://localhost:3000` to view the app.
